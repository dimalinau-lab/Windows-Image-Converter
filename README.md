# Windows Image Converter

Удобная утилита для быстрой конвертации изображений прямо из контекстного меню Windows. Программа работает в фоновом режиме, прячется в системном трее и поддерживает автозагрузку.

![Скриншот программы](screenshots/screen.png)

## 🚀 Возможности

* **Конвертация в 1 клик:** Добавляет пункты конвертации в классическое контекстное меню Windows (Правый клик).
* **Поддержка множества форматов:** PNG, JPEG, ICO, WEBP, BMP, GIF, TIFF.
* **Умная обработка:** Автоматически убирает прозрачный фон при конвертации в JPEG/BMP и правильно кадрирует иконки (ICO).
* **Системный трей:** Программа аккуратно висит возле часов и не мешает работе.
* **Автозагрузка:** Возможность включать/выключать старт вместе с Windows прямо из меню в трее.

## 📋 Системные требования

* **ОС:** Windows 10 / 11
* **Python:** Версия 3.8 или выше (требуется *только* для запуска из исходного кода).

## 🛠️ Установка и запуск

Вы можете установить программу двумя способами: скачать готовый файл или запустить скрипт через Python.

### Способ 1: Готовая программа (Рекомендуется)
Идеально подходит для тех, кто хочет просто пользоваться программой без установки Python.

1. Перейдите в раздел **[Releases](../../releases)** в этом репозитории.
2. Выберите и скачайте удобный формат:
   * **Установщик (Installer):** Обычный `.exe` файл для установки в систему.
   * **Портативная версия (Portable):** Просто распакуйте архив в удобное место.
3. Запустите программу. Она автоматически встроится в меню Windows и появится в трее.
