# Windows Image & AI Converter (WIC)

Удобная утилита для быстрой обработки файлов прямо из контекстного меню Windows. Программа работает в фоновом режиме, прячется в системном трее, умеет вырезать фон с помощью ИИ и конвертировать документы, абсолютно не мешая работе (работает без всплывающих окон консоли).

![Скриншот программы](screenshots/screen.png)

## 🚀 Возможности

* **Конвертация в 1 клик:** Добавляет пункты обработки в классическое контекстное меню Windows (Правый клик).
* **✨ Удаление фона (AI):** Автоматически вырезает фон с изображений с помощью встроенной нейросети прямо из меню.
* **📄 Работа с документами:** Быстрый перевод PDF в DOCX и DOCX в PDF.
* **Поддержка множества форматов изображений:** PNG, JPEG, ICO, WEBP, BMP, GIF, TIFF.
* **Умная обработка:** Автоматически убирает прозрачный фон при конвертации в JPEG/BMP и правильно кадрирует иконки (ICO).
* **Системный трей:** Программа аккуратно висит возле часов и не мешает работе.
* **Автозагрузка:** Возможность включать/выключать старт вместе с Windows прямо из меню в трее.

## 📋 Системные требования

* **ОС:** Windows 10 / 11
* **Python:** Версия 3.8 или выше (требуется *только* для запуска из исходного кода).

## 🛠️ Установка и запуск

Вы можете установить программу двумя способами: скачать готовый файл или запустить скрипт через Python.

### Способ 1: Готовая программа (Рекомендуется)
Идеально подходит для тех, кто хочет просто пользоваться программой без установки Python.

1. Перейдите в раздел **[Releases](../../releases)** в этом репозитории.
2. Выберите и скачайте удобный формат (Installer для установки или Portable для распаковки).
3. Запустите программу. Она появится в системном трее возле часов.
4. Кликните по синей иконке в трее правой кнопкой мыши и выберите **«Установить меню»**, чтобы кнопки конвертации появились в системе.

### Способ 2: Из исходного кода (Для разработчиков)
Так как программа использует тяжелые ИИ-библиотеки, запуск напрямую через Python обеспечивает максимальную скорость работы.

1. Склонируйте репозиторий командой: `git clone https://github.com/ВАШ_НИК/WIC-Converter.git`
2. Перейдите в папку проекта: `cd WIC-Converter`
3. Установите необходимые зависимости: `pip install -r requirements.txt`
4. Запустите программу: `python main.py`
5. Кликните по иконке в трее правой кнопкой мыши и выберите **«Установить меню»**.

## 🤖 Примечание по работе ИИ (Удаление фона)
При самом первом использовании функции удаления фона программе потребуется скачать веса нейросети `u2net.onnx` (~170 МБ). Это происходит автоматически и требует подключения к интернету. Все последующие удаления фона будут происходить моментально и полностью в оффлайн-режиме.
